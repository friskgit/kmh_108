Channel mapping plugin

// Channel mapping plugin that takes 8 channels of input (center speaker included) and maps it to the channel/speaker configuration of the studio 108 according to:

  • Audio interface ch -> signal output (clock wise starting at left)
  • 1 -> 1 (L)
  • 2 -> 3 (R)
  • 3 -> 9 (LSR)
  • 4 -> 4 (RSR)
  • 5 -> 8 (LSF)
  • 6 -> 5 (RSF)
  • 7 -> 7 (RL)
  • 8 -> 6 (RR)
  • 9 -> 2 (C)
