Channel mapping plugin

// Channel mapping plugin that takes 8 channels of input (center speaker excluded) and maps it to the channel/speaker configuration of the studio 108 according to:

  • Audio interface ch -> signal output (clock wise starting at left)
  • 1 -> 1 (L)
  • 2 -> 2 (R)
  • 3 -> 8 (LSR)
  • 4 -> 3 (RSR)
  • 5 -> 7 (LSF)
  • 6 -> 4 (RSF)
  • 7 -> 6 (RL)
  • 8 -> 5 (RR)
